
# PDF System — Premium Minimal Layout

PDF rendering uses:
- **Satori** — JSX → SVG converter
- **Resvg** — PNG/PDF renderer

### Features:
- Modern minimal Apple-style layout
- Clean typography: Inter font
- Cover page
- Summary page
- Triad Engine details:
  - Palmistry table
  - Astrology table
  - Numerology table
- Footer logo + contact details

A full rendering pipeline example is provided in the codebase.
