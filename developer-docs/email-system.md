
# Email System (Resend)

The platform uses **Resend** for:

- Transactional emails
- PDF attachments
- HTML templates

### Features:
- Premium minimal template
- “Report Ready” emails
- Personal or Technical report distinction
- Auto-detection of missing emails
- Support for future email digests

### Required Variables:
- RESEND_API_KEY

See `.env.example` for more.
