
# Environment Variables (Full List)

## Required for production:
- **OPENAI_API_KEY** — AI model access  
- **RESEND_API_KEY** — Email delivery  
- **RECAPTCHA_SECRET_KEY** — Server validation of reCAPTCHA  
- **APP_DOMAIN** — URL of production Shopify store  
- **SUPPORT_EMAIL** — official sender email (Resend verified)

## Optional (future features):
- STRIPE_SECRET_KEY
- STRIPE_WEBHOOK_SECRET
- AWS_S3_BUCKET
- LOGGING_TOKEN
